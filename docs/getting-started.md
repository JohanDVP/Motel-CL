# Getting Started

Esta guía explica cómo instalar y ejecutar el proyecto **Motel-CL** en tu entorno local.



## Requisitos

- Python 3.10 o superior
- `uv` (gestor de dependencias moderno)
- Git



## Instalación del proyecto

Clona el repositorio:

```bash
git clone https://github.com/JohanDVP/Motel-CL
cd Motel-CL
